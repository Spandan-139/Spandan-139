# Hi, I'm Spandan 👋

I'm a B.Tech CSE student at SRMIST building production-style ML systems — the kind that actually run in the real world, not just notebooks.

My focus is **ML Engineering, Data Engineering, and LLM/RAG systems**. I care about the full stack: clean pipelines, low-latency serving, and systems that don't silently break.

Currently open to **SWE / ML Engineering internships**.

---

## 🔨 What I'm Building

**[feature-forge](https://github.com/Spandan-139/feature-forge)** — A production-style ML Feature Store inspired by Uber's Michelangelo and Google's Feast. Built from scratch with offline/online serving, a model registry, and a real-time monitoring dashboard. Stack: Polars, DuckDB, Redis, FastAPI, MLflow, Docker. Dataset: 2.7M NYC Taxi trips.
